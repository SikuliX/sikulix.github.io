What is it?

SikuliX automates anything you see on your Windows, Mac or Linux screen. It uses image recognition powered by OpenCV.

Automate Visual Workflows

Mimic user behaviour on any user interface on your screen. Including automating keyboard and mouse actions.

Fastest Visual Testing

SikuliX IDE lets you setup, maintain and automate visual workflows. Easily capture and organize the needed images.

Use Cases

Automating repetitive daily tasks on applications, websites, games, administration of IT systems and networks.

Supported Languages

You can script in SikuliX using any of: JavaScript, Python, Ruby, Java, Scala, Clojure or RobotFramework.

Maintained by Community

SikuliX is a 100% open-source and exists only due to its community ❤. If you like it, consider contributing back!
